## TaskRoom
O desafio consiste em fazer uma aplicação web para cadastro de usuário e gerenciamento de tarefas (Adicionar, listar, editar, excluir e filtrar tarefas)

### Arquitetura de Pastas:

##### Backend:
```plaintext
├── src
│   ├── controllers
│   ├── middlewares
│   ├── models
│   ├── routes
│   └── services
├── tests
├── .env
├── .gitignore
├── package.json
└── tsconfig.json
```

##### Frontend:
```plaintext
├── src
│   ├── assets
│   ├── components
│   └── services
│   ├── App.tsx
│   ├── main.tsx
├── index.html
├── .gitignore
├── package.json
├── tsconfig.json
└── vite.config.ts
```
